--- title: "Soumettre la documentation" subtitle: "Vous avez un bon tutoriel que vous aimeriez partager?" slug: soumettre permalink: /fr/share/ lang: fr --- La plateforme de documentation du MSdS MiData est conçue pour être modifiée par toi!

Si tu vois quelque chose qui te dérange ou s'il te manque une information dans la documentation, tu peux la modifier toi-même. N'hésite pas à optimiser les textes, à ajouter des images ou même à créer de nouvelles pages avec des sujets qui n'ont pas encore été traités. Nous nous réjouissons de toute contribution qui permet d'améliorer la documentation.

Le point de départ des modifications est le GitHub Repository, où est hébergée cette plate-forme de documentation. On y trouve tous les contenus, ainsi que le code source du site web. Pour pouvoir proposer des modifications au contenu, tu dois avoir une compréhension de base de Git, GitHub et de la syntaxe Markdown.

Le lien suivant explique comment tu peux contribuer au contenu ou au code de la plate-forme de documentation PBS MiData : Contributing to MiData docu (EN).
